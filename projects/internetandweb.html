Internet

It is a worldwide system which connects computers and networks that allows you to share information to standard communication rules.

Web

It is commonly define as the worldwide web or "web" is one service that runs the internet. It allows people to access and share information using the browsers and links

How are the Internet and Web different?

The internet connects everything such as the wires, sattelites and servers. The Web is a tool that uses the internet to share documents, media and sites.

ASCII and Unicode

ASCII is an older coding system that represents English letters, symbols and numbers from 0-127. While on the other hand the Unicode is a newer system which supports thousands of characters from many world languages and symbols.

What is UTF-8

UTF-8 is a way to store Unicode characters using a flexible length. It uses one to four bytes per character which makes it efficient for English and can still able to handle it global languages.

URL

A Uniform Resource Locator is the address you type in a browser to reach a website or an online resource.

Domain Name

It is something that you can easily remember let say like "google.com" that points to a website.

HTTP

Hypertext Transfer Protocol is the system that define how web browsers and servers send and receive images, web pages and other resources.

HTTPS

It is a much secure version of HTTP. This one adds encryption to make sure that the information exchanged between the browsers and servers are private and safe from tampering.

IP Address

An IP Address is a unique number that identifies each device on a network. It acts like a digital home address to ensure that the information is delivered to the right computer or server.

DNS

The Domain Name System is like a phonebook of the internet. It can translates human friendly domain names like byuh.edu into machine friendly IP Addresses that a computer use to connect.

W3C Recommendations

These are the standards created by the World Wide Web Consortium. They guide how web technologies should be be built so that websites work consistently accross to different browsers and devices.

XML, XHTML, HTML, HTML5

HTML: This is the basic language used to create web pages, to define structure with tags. XHTML: A stricter version of HTML that follows XML rules. XML: It is a flexible maekup language for storing and transporting data, not just for web pages. HTML5: This is the latest version of HTML, with new features like video, audio and to better support for mobile devices.

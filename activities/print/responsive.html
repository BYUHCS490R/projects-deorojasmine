Answer to the Engagement Today: Monday, October 27th

Engagement Today

1. Why do we need a viewport meta tag? The viewport meta tag makes your webpage fit and look right on phones and tablets.

2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority? The last CSS rule written takes priority if both target the same tag.

3. How do you change the CSS for a web page displayed on a phone? You change CSS for phones using a media query like @media (max-width: 600px) { ... }.

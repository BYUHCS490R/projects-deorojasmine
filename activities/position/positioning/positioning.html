Engagement Answers today 1. What is normal flow in HTML? Normal flow is the default way elements are arranged on a webpage block elements stack top to bottom, and inline elements go left to right. 2. What does float do? Why do you need to clear the float? The float property lets elements move to the left or right, allowing text or content to wrap around them; you “clear” the float so that other elements don't accidentally overlap or wrap beside it. 3. What do absolute, fixed, and relative do? These are CSS positioning types relative moves an element slightly from its normal spot, absolute positions it based on its nearest positioned parent, and fixed keeps it in place even when scrolling. 4. How can you make nice navigation buttons for mobile devices? You can use simple CSS styling with padding, colors, and hover effects, or use responsive tools like Flexbox or media queries to make buttons adjust to smaller screens. 5.
